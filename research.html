Honours Project

Undergraduate Thesis | University of Ottawa

End-to-End Sentiment Classification: From Model Design to Production Deployment

Abstract

This thesis presents a comprehensive study of sentiment classification systems, addressing both the modeling challenge of extracting sentiment from text and the engineering challenge of deploying such systems at scale. The work spans the complete machine learning lifecycle: data collection and preprocessing, model architecture selection and fine-tuning, rigorous evaluation methodology, and production deployment with monitoring.

Research Questions
  • How do pre-trained transformer models compare to traditional ML approaches for domain-specific sentiment classification?
  • What is the relationship between model complexity, inference latency, and classification performance?
  • How can attention mechanisms be leveraged to provide interpretable predictions for business stakeholders?
  • What MLOps practices are essential for maintaining model quality in production?
Key Contributions
  • Methodological: Developed a systematic framework for comparing transformer architectures on sentiment tasks, including statistical significance testing and effect size analysis
  • Technical: Implemented model distillation techniques that reduced inference time by 4x while maintaining 95% of original accuracy
  • Practical: Built a complete MLOps pipeline with automated retraining, A/B testing infrastructure, and model monitoring dashboards
  • Interpretability: Created attention visualization tools that highlight which words most influence classification decisions
Results

The fine-tuned RoBERTa model achieved a 92% F1-score on the held-out test set, outperforming the SVM baseline by 12 percentage points. The distilled model reduced inference latency from 150ms to 35ms while maintaining 87% F1-score, making it suitable for real-time applications.

Optimization & Theory

Foundational experiments in machine learning optimization

Gradient Descent Convergence Analysis
Motivation

Understanding why optimization algorithms converge (or fail to) is essential for debugging training issues and selecting appropriate hyperparameters. This study provides empirical validation of theoretical convergence results.

Methodology

Implemented batch gradient descent, stochastic gradient descent, mini-batch SGD, and Adam from scratch. Analyzed convergence behavior on:

  • Convex quadratic functions (analytical solution available)
  • Logistic regression (convex but non-quadratic)
  • Neural networks (non-convex)
Key Findings
  • Verified theoretical O(1/t) convergence rate for convex problems
  • Documented the variance-reduction effect of increasing batch size
  • Identified pathological loss landscapes where Adam outperforms SGD significantly

Loss Landscape Visualization
Motivation

The geometry of loss landscapes directly affects training dynamics. Visualizing these landscapes provides intuition about why certain architectures are easier to train than others.

Methodology

Used filter-wise normalization to create 2D slices of high-dimensional loss surfaces. Compared landscapes for:

  • Networks with/without skip connections
  • Different activation functions (ReLU, tanh, sigmoid)
  • Varying depth and width configurations
Key Findings
  • Skip connections create "flatter" minima with better generalization
  • Batch normalization smooths the loss landscape significantly
  • Deeper networks have more complex loss geometry without residual connections
